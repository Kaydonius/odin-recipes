# odin-recipes

This current project involves me creating a website with links to 3 recipes. The recipes should all have an image, description, list of ingredients and steps to complete. Each recipe page should also have a link going back to the homepage.

By completing this project, I gain a better understanding of links, images, and lists in html. I also gain a familiarity with using git in the command line, understanding file structure and how it correlates to websites, and a little bit of visual organization for aesthetic appeal.
